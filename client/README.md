# kreditpay frontend

This submodule contain the kreditpay frontend logic.

## Quick Setup

```bash
## Install dependencies
$ npm install

## Create env file
$ cp env.sample .env

## Run Server
$ npm run dev

## Create Build
$ npm run build
```

## Environment Variable Setup

Below is the default config

```
API_LOCATION=http://localhost:3000/
```

You can change this according to your need.
